# 🚀 Dynamic Multi-Template Resume Builder

A modern, Canva-inspired Resume Builder application built using **React + Vite + Tailwind CSS**, featuring real-time editing, dynamic template rendering, professional PDF export, and adaptive resume layouts.

🌐 **Live Website:**
https://resume-builder-three-snowy.vercel.app/

---

# 📌 Overview

This project is a fully dynamic resume generation platform designed to help users create highly professional resumes with real-time editing and multiple premium template options.

Unlike traditional resume builders, this application dynamically adapts both the editor and preview architecture depending on the selected template, creating a true Canva-style editing experience.
